## üèóÔ∏è Technical Architecture

### Core Components

1. **Interactive Dashboard** (`app/dashboard.py`)
   - Streamlit-based real-time UI
   - 5 comprehensive tabs (Overview, Analysis, AI Detection, Geographic, Analytics)
   - Live threat monitoring and visualization
   - Customizable filters and alerts

2. **Threat Detection Engine** (`app/threat_detector.py`)
   - Multi-layered security analysis
   - IOC database integration
   - YARA-style detection rules
   - Network traffic analysis
   - Malware detection
   - Phishing analysis
   - Behavioral anomaly detection

3. **AI/ML Models** (`app/ml_models.py`)
   - Isolation Forest for anomaly detection
   - Random Forest for threat classification
   - LSTM autoencoders
   - Behavioral analytics engine

4. **Security Framework** (`utils/security.py`)
   - AES-256 encryption
   - JWT authentication
   - RBAC implementation
   - Data anonymization (GDPR compliant)
   - Audit logging

### Infrastructure

- **Database**: PostgreSQL for persistent storage
- **Cache**: Redis for high-performance caching
- **Search**: Elasticsearch for log analysis
- **Monitoring**: Prometheus + Grafana
- **Deployment**: Docker Compose orchestration
